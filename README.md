# Badplatser i Sundsvall
Länk till webbsida: [Badplatser i Sundsvall](https://frontendprojektjoba1602.netlify.app/)
Denna webbsida hämtar information via öppen data från Sundsvalls kommun, mer specifikt data om badplatserna inom kommunen. Denna information består av badplatsens namn, koordinater, beskrivning samt temperaturmätning på vissa av badplatserna. Webbsidan jag skapat placerar ut markörer med hjälp av leaflet och koordinat-data från API:et. Med varje individuell badplats medföljer information för användaren att läsa vid klick av markör på kartan.
